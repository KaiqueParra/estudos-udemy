Posicionamentos

Parágrafo 1, este é um exemplo de um elemento com posicionamento static. O posicionamento static é o padrão para todos os elementos HTML. Isso significa que eles são posicionados de acordo com o fluxo normal do documento, sem qualquer ajuste especial.

Parágrafo 2, este é mais um exemplo de um elemento com posicionamento relative. O posicionamento relative permite que você mova o elemento em relação à sua posição original, sem retirá-lo do fluxo normal do documento.

Parágrafo 3, este é mais um exemplo de um elemento com posicionamento absolute. O posicionamento absolute remove o elemento do fluxo normal do documento e o posiciona em relação ao seu ancestral mais próximo que tenha um posicionamento diferente de static.

Parágrafo 4, este é mais um exemplo de um elemento com posicionamento fixed. O posicionamento fixed é frequentemente usado para criar elementos que permanecem visíveis na tela, como barras de navegação fixas, cabeçalhos ou rodapés. No entanto, é importante ter cuidado ao usar o posicionamento fixed, pois ele pode interferir na experiência do usuário, especialmente em dispositivos móveis.
